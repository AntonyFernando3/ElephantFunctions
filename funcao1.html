PARA QUE SERVE A FUNÇÃO STRTOUPPER() ?


A função STRTOUPPER() no PHP é usada para converter uma string para letras maiúsculas. Ela recebe uma string como parâmetro e retorna uma nova string com todas as letras em maiúsculas. Por exemplo, se você tiver a string "Olá, mundo!", ao passá-la como parâmetro para a função STRTOUPPER(), ela retornará a string "OLÁ, MUNDO!".

Exemplo:


$string = "Exemplo de Texto";
$string_uppercase = strtoupper($string);
echo $string_uppercase; // saída: "EXEMPLO DE TEXTO"
