PARA QUE SERVE A FUNÇÃO IS_BOOL() ?


A função em PHP é usada para verificar se uma determinada variável é do tipo booleano ou não. Ela retorna se a variável for do tipo booleano e caso contrário.is_booltruefalse
O tipo booleano representa um valor lógico verdadeiro ou falso, ou seja, apenas dois possíveis valores. É comumente utilizado em expressões condicionais, onde uma decisão deve ser tomada com base em uma condição booleana.

Exemplo:


$a = false;
$b = 0;

// Como $a é boolean, retornará true
if (is_bool($a) === true) {
echo 'Sim, isso é booleano';
}

// Como $b não é boolean, retornará false
if (is_bool($b) === false) {
echo 'Não, isso não é booleano';
}
?>

Aqui possuimos um programa bem simples utilizando a função Is-bool no php.

if (isset($_GET['ativo'])) { // verifica se o parâmetro 'ativo' está definido na URL
$ativo = $_GET['ativo']; // recupera o valor do parâmetro 'ativo'

if (is_bool($ativo)) { // verifica se o valor do parâmetro é do tipo booleano
if ($ativo) {
echo "O parâmetro 'ativo' é verdadeiro";
} else {
echo "O parâmetro 'ativo' é falso";
}
} else {
echo "O valor do parâmetro 'ativo' não é um booleano válido";
}
} else {
echo "O parâmetro 'ativo' não foi definido na URL";
}

Nesse exemplo, a função isset é usada para verificar se o parâmetro 'ativoativo está definido na URL. Em seguida, o valor do parâmetro é recuperado da URL e armazenado na variável '$ativo$ativo. A função is_bool é usada para verificar se o valor do parâmetro é do tipo booleano. Se for, o código verifica se é true ou false. Se não for do tipo booleano, o código exibe uma mensagem de erro. Se o parâmetro ativo não estiver definido na URL, o código exibe uma mensagem informando isso.
