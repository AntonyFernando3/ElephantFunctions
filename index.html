O QUE É A ELEPHANT FUNCTIONS?

Elephant Functions é uma plataforma 100% online que oferece exemplos de 4 tipos de funções na linguagem de programação PHP.


PORQUE A LOGO DA PLATAFORMA É UM ELEFANTE?

O PHP tem esse mascote, por causa da sua sigla. Ele foi criado pelo designer Vincent Pontier, que comercializa as produtos oficiais do elefantinho, em sua loja virtual. Vincent Pontier achou que as letras da sigla PHP, olhadas de um certo angulo, pareciam um elefante.
Devido a esse motivo, procuramos dar grande relevancia ao PHP trazendo seu mascote como logo da plataforma. Há boatos de muitas piadas alegando que o PHP tem um símbolo como elefante alegando que a linguagem é pesada e lenta, por isso um elefante. Mas, isso é apenas uma brincadeira, pois, a velocidade do PHP depende da configuração do seu arquivo como qualquer outro tipo de linguagem de programação.


QUEM FUNDOU A PLATAFORMA ELEPHANT FUNCTIONS

A elephant functions foi fundada pelos alunos da Faculdade de Tecnologia de Jundiaí (FATEC) pertencente ao Centro Paula Souza do Governo do Estado de São Paulo. Antony Fernando Ribas Rocha Silva, Matheus Henrique Santos e Silva, Gabriele Xavier Ferreira e Thais Priscila Lopes criaram a plataforma nas aulas de Linguagem de Programação sob instrução da professora Luciana com o intuito de criar uma plataforma que oferecesse exemplos de Funções em linguagem de programação PHP.


QUAIS FUNÇÕES A PLATAFORMA OFERECE EXEMPLOS?

Atualmente, a nossa plataforma oferece exemplos de 4 tipos de funções em Linguagem de Programação PHP, entre elas a Função 1 - Strtoupper(), Função 2 - Strtolower(), Função 3 - Empty() e Função 4 - Is-bool().

Função 1 - Strtoupper()

A função Strtoupper() no PHP é uma função embutida (built-in) que converte uma string (cadeia de caracteres) para letras maiúsculas. Ela retorna uma nova string com todas as letras em maiúsculas.

A sintaxe básica da função é a seguinte:

strtoupper(string $string): string

Função 2 - Strtolower()

A função Strtolower() no PHP é uma função embutida (built-in) que converte uma string (cadeia de caracteres) para letras minúsculas. Ela retorna uma nova string com todas as letras em minúsculas.

A sintaxe básica da função é a seguinte:

strtolower(string $string): string

Atualizado em Março de 2023

Função 3 - Empty()

A função empty() é uma função embutida (built-in) do PHP que verifica se uma variável está vazia ou não. Ela retorna true se a variável estiver vazia, e false caso contrário.



A sintaxe básica da função é a seguinte:

empty(mixed $var): bool

Atualizado em Março de 2023

Função 4 - Is_bool()

A função is_bool() é uma função embutida (built-in) do PHP que verifica se uma variável é do tipo booleano ou não. Ela retorna true se a variável for do tipo booleano, e false caso contrário.

A sintaxe básica da função é a seguinte:

is_bool(mixed $var): bool
